Ce plugin vous permet d'ajouter facilement un slider sur la page d'accueil de votre blog. Vous pouvez aussi choisir de le mettre en place sur d'autres pages, à l'emplacement le plus approprié.

Mode d'emploi

  1. Activer le plugin (onglet Paramètres)
  2. Ajouter des images

Mise en place

Vous avez plusieurs possibilités pour mettre en place le contenu généré par le plugin.

Automatiquement

Le contenu est mis en place automatiquement en utilisant le comportement publicTopAfterContent, uniquement sur la page d'accuei.

En ajoutant un tag dans un des templates de votre thème

Le plugin définit un nouveau tag dcSlider

A l'emplacement souhaité, insérer le code suivant:

{{tpl:dcSlider}}
