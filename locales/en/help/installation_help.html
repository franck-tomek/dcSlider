That plugin allows you to add a slider on the homepage of you blog. You can also add it in another page, at the right place.

Usage

  1. Activate the plugin (tab Settings)
  2. Add images

Installation

You can install it in many ways.

Automatically

The content can be automatically added by using the behavior publicTopAfterContent, on the homepage only.

By modifying a template of your theme and adding a tag

The plugin define a new tag dcSlider

Choose the right place and add the following code in your template:

{{tpl:dcSlider}}
